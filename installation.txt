WP SENTINEL – INSTALLATION GUIDE

Version: 4.6
Developer: Ajaypal Singh

INSTALLATION STEPS:

Step 1:
Login to WordPress Admin Dashboard

Step 2:
Go to:
Plugins → Add New

Step 3:
Click:
Upload Plugin

Step 4:
Select:
ironwall.zip

Step 5:
Click:
Install Now

Step 6:
Click:
Activate Plugin

Step 7:
Go to:
Security → Settings

Step 8:
Enable recommended protections:
✓ Login Protection
✓ XMLRPC Protection
✓ Security Headers

Step 9:
Set custom login URL
Example:
secure-login

IMPORTANT:
Bookmark your custom login URL before saving.

Example:
https://yourwebsite.com/secure-login

INSTALLATION COMPLETE.
