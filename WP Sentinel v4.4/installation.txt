WP Sentinel v4.4 - Installation Guide
=======================================

Thank you for choosing WP Sentinel! This plugin provides enterprise-grade Web Application Firewall (WAF), advanced malware scanning, dual-factor authentication, and rate-limiting to protect your WordPress site.

Requirements:
- WordPress 5.8 or higher
- PHP 7.4 or higher
- MySQL 5.6 or higher

Installation Steps:
1. Log in to your WordPress Admin dashboard.
2. Navigate to "Plugins" -> "Add New".
3. Click "Upload Plugin" at the top of the screen.
4. Select the `wp-sentinel.zip` file provided in this release folder.
5. Click "Install Now".
6. After installation completes, click "Activate Plugin".

Post-Installation:
1. Navigate to the new "WP Sentinel" menu item in your left-hand admin sidebar.
2. Go to "Settings" to enable core protections.
3. We strongly recommend enabling "Prevent Brute Force Attacks" and "Enable 2FA".
4. Go to "Scanner" and run your first full site scan to baseline your security.
